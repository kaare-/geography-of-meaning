# Culture

> **Guiding question:** How do shared predictive patterns stabilize in populations?

## Status

**Stub** — long-timescale population dynamics; deferred. Distinct from [13_inheritance.md](13_inheritance.md) (generational transfer of structure). Builds on inheritance, [10_communication.md](10_communication.md), and [15_information_storage_and_external_memory.md](15_information_storage_and_external_memory.md).

## Summary

Culture is **population-stable predictive structure** — shared compressed concepts, traditions, and trace patterns that outlive any individual. Never pre-programmed as myths, stories, or knowledge systems.

## Planned

- Inherited concepts via genome bias + social learning (no graph copy)
- Shared concepts stabilized across signatures
- Traditions as repeated sound→action→outcome chains
- Stories as temporal activation sequences ([10_communication.md](10_communication.md))
- Myths as high-confidence low-falsifiability concept clusters (researcher label)
- Knowledge systems as landscape + memory + communication coupling
